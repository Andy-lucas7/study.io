# 📚 study.io

**Study.io** é um app Flutter para te ajudar a estudar de forma mais eficiente!  
Ele inclui técnicas como Pomodoro, gerenciamento de tarefas, criação de resumos com voz, rastreamento de progresso e sons ambientes com temas dinâmicos.

## ✨ Funcionalidades

- ⏱️ Timer Pomodoro para ciclos de foco.
- 📝 Criação de resumos por voz.
- 📊 Página de progresso para acompanhar seu desempenho.
- ✅ Gerenciamento de tarefas locais (offline).
- 🎵 Sons ambientes com temas personalizados (chuva, floresta, etc).
- 🌙 Tema escuro adaptável ao som escolhido.
- 📂 Salvamento local com JSON/SQLite.
